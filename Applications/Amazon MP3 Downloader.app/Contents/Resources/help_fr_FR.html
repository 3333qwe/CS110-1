Prérequis du navigateur

Bien que les titres de la boutique Amazon MP3 soient compatibles avec les navigateurs les plus couramment utilisés, il se peut que vous deviez configurer les paramètres de votre navigateur et de votre système de manière à permettre aux téléchargements en provenance d'Amazon.fr de s'effectuer avec succès.

Effectuer des achats via votre navigateur par défaut
L'Amazon MP3 Downloader fonctionne avec le navigateur par défaut de votre système. Il se peut que vous rencontriez des problèmes pour démarrer ou terminer vos téléchargements si vous utilisez un autre navigateur que votre navigateur par défaut pour effectuer vos achats sur Amazon MP3.

Par exemple, si vous installez l'Amazon MP3 Downloader à l'aide d'un navigateur autre que votre navigateur par défaut (par exemple Firefox alors que votre navigateur par défaut est Internet Explorer), votre navigateur par défaut s'ouvrira une fois le Downloader installé, et vous ne pourrez pas effectuer d'achats par le biais d'un autre navigateur.

Pour effectuer vos achats à l'aide d'un autre navigateur :

  • Paramétrez votre navigateur préféré comme navigateur par défaut : merci de consulter la section d'Aide de votre navigateur pour obtenir des informations quant à son installation comme navigateur par défaut. Une fois votre navigateur paramétré, installez l'application MP3 Downloader.

  • Autorisez l'installation du Downloader sur votre navigateur actuel : si le Downloader est déjà installé dans votre système, visitez la page http://www.amazon.fr/gp/dmusic/help/amd.html depuis votre navigateur préféré et suivez les instructions données dans la section « Besoin d'Aide ? ».

Ajuster vos paramètres
Dans certains cas, les paramètres de votre navigateur peuvent empêcher le téléchargement de fichiers vers votre ordinateur. Merci de vous référer à la section d'Aide de votre navigateur pour des instructions détaillées sur le paramétrage.

  • Autoriser les cookies : le Downloader MP3 utilise des cookies en provenance d'Amazon.fr pour confirmer vos achats et organiser le téléchargement de fichiers MP3. Il vous faudra autoriser les cookies provenant d'Amazon.fr pour que vos téléchargements soient possibles.

    Si vous n'autorisez pas les cookies ou les autorisez seulement de manière temporaire pour une session donnée, il se peut que l'on vous demande d'installer le Downloader après chaque achat. Vous règlerez ce problème en acceptant systématiquement les cookies en provenance d'Amazon.fr.

  • Désactiver les bloqueurs de pop-ups : votre navigateur doit être paramétré de manière à autoriser les fenêtres pop-ups provenant d'Amazon.fr. Le Downloader et votre navigateur par défaut ouvriront de nouvelles fenêtres pour vous aider dans l'achat et le téléchargement des fichiers MP3.

  • Autoriser les téléchargements en provenance d'Amazon.fr : il se peut que vous ayez à ajouter Amazon.fr à la liste des sites de confiance, ou à autoriser les téléchargements en provenance d'Amazon.fr si votre navigateur affiche un message d'alerte relatif à la sécurité.

  • Autoriser les notifications : dans certains cas, les paramètres de sécurité de votre navigateur peuvent empêcher le téléchargement de certains fichiers sans vous avertir qu'une tentative de téléchargement est en cours, en particulier si vous utilisez Internet Explorer. Vérifiez les paramètres de sécurité de votre navigateur afin d'autoriser les messages d'alerte automatiques relatifs aux téléchargements de fichiers de manière à être informé si votre navigateur est en train d'empêcher un téléchargement.

Installation du Downloader

Lors de votre premier achat de MP3, il vous sera automatiquement proposé d'installer l'application Amazon MP3 Downloader.

Pour installer l'Amazon MP3 Downloader :

  1. Cliquez ici pour installer le Downloader.
  2. Cliquez sur le lien ou le bouton correspondant à votre système d'exploitation pour télécharger l'application.
  3. Suivez les instructions qui vous seront données pour installer l'application.

Tester le Downloader Amazon MP3 avec votre titre gratuit
Une fois terminée l’installation du Downloader, nous vous offrons un titre gratuit pour que vous puissiez le tester et vous assurer qu’il fonctionne. Si vous installez le Downloader lors de votre premier téléchargement de MP3, la musique que vous achetez sera téléchargée après le morceau gratuit.

Assurez-vous de cliquer sur “Ouvrir” quand l’option vous est proposée pour le fichier .amz, et de vérifier que votre Downloader est paramétré de manière à toujours être utilisé pour ouvrir les fichiers .amz. Les fichiers de type .amz activent le Downloader et démarrent les téléchargements MP3.

Lorsque votre titre gratuit sera téléchargé dans son intégralité, le Downloader Amazon MP3 ouvrira votre lecteur iTunes ou Windows Media Player (celui qui est installe) et importera automatiquement le morceau téléchargé pour que vous puissiez l’écouter. Voir « Import des titres vers iTunes ou Windows Media Player ».

Utilisation du Downloader

La musique que vous achetez sur Amazon MP3 doit être téléchargée au moment de l'achat. L'application Amazon MP3 Downloader simplifie le téléchargement et le rangement de la musique que vous achetez en permettant de télécharger des albums entiers, de placer plusieurs titres à la suite les uns des autres et d'importer automatiquement de la musique vers iTunes et Windows Media Player.

Démarrer le Downloader
Le Downloader se déclenche automatiquement à chaque fois que vous effectuez un achat. Cependant, vous devrez le démarrer manuellement depuis votre ordinateur si vous faites une pause lors du téléchargement ou si vous l'annulez, ou si vous sauvegardez votre fichier .amz sur votre ordinateur au lieu de l'ouvrir avec le Downloader.

Note : les téléchargements sont disponibles pour une courte période après achat. Si vous avez sauvegardé votre fichier .amz sur votre ordinateur, nous vous conseillons de cliquer sur « Ouvrir » dans le Downloader immédiatement après la sauvegarde. Si vous attendez avant d'effectuer cette démarche, votre téléchargement pourrait ne plus être valide et devenir indisponible.

Voici où trouver le Downloader si vous avez besoin de redémarrer un téléchargement incomplet :

  • Windows : Démarrer > Tous les programmes > Amazon > Amazon MP3 Downloader
  • Mac : sélectionnez « Amazon MP3 Downloader » dans le dossier « Applications ».
  • Linux : l'application est installée sous le nom /usr/bin/amazonmp3 et une icône apparaît dans le menu « Applications » dans le sous-menu « Internet ».

De manière générale cependant, nous vous recommandons de ne pas faire de pauses, sauf si vous pouvez reprendre le téléchargement dans les 3 heures qui suivent. Si vous rencontrez des difficultés pour reprendre le téléchargement après une pause, merci de contacter notre Service Client.

Import des titres vers iTunes ou Windows Media Player

Si iTunes ou Windows Media Player sont installés sur votre ordinateur, l'Amazon MP3 Downloader reconnaîtra automatiquement l'application en question comme votre lecteur MP3 et importera votre musique dans votre bibliothèque. Si aucune de ces applications n'est installée, le Downloader sauvegardera vos MP3 dans votre dossier de téléchargement par défaut.

Note : la fonction d'import dans la bibliothèque n'est pas disponible pour la version Linux du Downloader.

Pour paramétrer l'import par défaut :

  1. Démarrer l'application Amazon MP3 Downloader.
  2. Sélectionner Fichier > Préférences dans le menu du Downloader.
  3. Choisissez le lecteur par défaut dans le menu déroulant en-dessous de « Lorsque le téléchargement d'un morceau touche à sa fin… », ou sélectionnez « Ne rien faire » si vous ne souhaitez pas utiliser la fonction d'import.

Les options « l'ajouter à iTunes » ou « l'ajouter à Windows Media Player » apparaîtront uniquement si une de ces applications est installée sur votre ordinateur.

Note relative à l'utilisation d'iTunes
Dans de rares cas, il arrive que le Downloader ne puisse pas importer les titres vers iTunes en raison de potentiels problèmes avec la version d'iTunes installée. Le Downloader devrait alors afficher un message sur la droite de votre écran vous informant de cet échec. Si cela devait se produire, nous vous recommandons de vous rendre sur www.iTunes.fr et d'installer la dernière version d'iTunes en date. Après avoir réinstallé iTunes, veuillez redémarrer votre ordinateur, démarrer iTunes, et accepter les conditions d'utilisation d'Apple avant de pouvoir utiliser l'Amazon MP3 Downloader pour importer des titres vers iTunes.

Vous pouvez également envisager d'ajouter votre fichier de téléchargement dans vos préférences iTunes. Pour cela, démarrez iTunes et sélectionnez l'option  « Ajouter le dossier à la bibliothèque » dans le menu, puis sélectionnez le fichier dans lequel vos titres sont téléchargés depuis Amazon MP3. Le dossier par défaut s'appelle « Amazon MP3 » et est situé dans « Ma musique » sous Windows ou « Musique » sous Mac.

Localisation des titres téléchargés

Lorsque l'Amazon MP3 Downloader sauvegarde des titres dans votre ordinateur, ils sont enregistrés par défaut dans un dossier appelé « Amazon MP3 ». Ce dossier se trouve dans « Ma musique » sous Windows ou dans « Musique » sous Mac. Nous vous recommandons d'utiliser ce dossier par défaut, mais vous avez la possibilité d'en changer si vous le souhaitez.

Si vous n'utilisez pas le Downloader pour sauvegarder votre musique, vous pouvez retrouver les fichiers en recherchant « .mp3 » sur votre ordinateur ou en vérifiant là où votre navigateur stocke les dossiers par défaut.

Pour localiser vos vidéos dans Windows Media Player, cliquez sur la flêche pointant vers le bas en dessous de la « Bibliothèque » en haut au centre de l'application et sélectionnez « vidéo » dans le menu déroulant. Vous devriez maintenant voir les vidéos téléchargées à l'aide du Downloader Amazon MP3. Les livrets numériques ne sont pas téléchargés dans Windows Media Player mais dans votre dossier de téléchargement par défaut.

Pour modifier l'endroit où votre navigateur stocke ces fichiers par défaut :

  1. Démarrez l'Amazon MP3 Downloader.
  2. Sélectionnez « Fichier > Préférences » dans le menu du Downloader.
  3. Cliquez sur le bouton Modifier dans la section « Répertoire de musique ».
  4. Sélectionnez un dossier dans la liste.
  5. Si vous souhaitez créer un sous-dossier dans le dossier sélectionné, cliquez sur Créer un nouveau dossier.
  6. Cliquez sur OK lorsque vous avez terminé.

Options dans le menu du Downloader

Les options suivantes sont disponibles dans le menu du Downloader :

  • Fichier > Préférences : ouvre la fenêtre de Préférences à partir de laquelle vous pouvez adjuster les paramètres du Downloader. Pour de plus amples informations, consulter la partie « Modification des paramètres du Downloader » ci-dessous."

  • Fichier > Quitter : ferme le Downloader.

  • Aide > Afficher l'aide : ouvre la page d'Aide relative au Downloader sur une nouvelle page du navigateur.

  • Aide > Rechercher les mises à jour : vérifie sur Amazon.fr si une nouvelle version du Downloader est disponible.

  • Help/Contact Customer Service: Customer Service: Opens the Customer Service contact us page.

  • A propos de : affiche les informations sur la version du Downloader utilisée.

Modification des paramètres du Downloader

Pour accéder aux Préférences du Downloader, choisissez « Fichier » dans le menu puis cliquez sur Préférences. Voici une liste des paramètres que vous pouvez modifier :

  • Répertoire de musique : cette option vous permet de modifier l'endroit où vous souhaitez recevoir vos téléchargements. Cliquez sur Modifier pour choisir ou créer un nouveau dossier. Cliquez sur OK quand vous avez terminé.

  • Bibliothèque : vous pouvez grâce à cette option sélectionner dans le menu déroulant la bibliothèque dans laquelle vous souhaitez que vos téléchargements soient automatiquement classés. Pour l'instant, cette option est disponible pour iTunes et Windows Media Player. Si vous choisissez « ne rien faire », vos fichiers seront stockés dans le dossier « Amazon MP3 » et peuvent être ajoutés manuellement à la librairie de votre choix. Cette option n'est pas disponible pour Linux.

  • Rechercher automatiquement les mises à jour du programme : si vous cliquez dans la case de cette option, le Downloader vérifiera systématiquement si des mises à jour sont disponibles lorsque vous ouvrirez l'application.

  • Réduire la zone de notification (pour Windows uniquement) : cette option vous permet d'effectuer une réduction de l'application dans la barre d'état système. Si vous activez cette option, vous pourrez également autoriser les info-bulles qui affichent l'état d'avancement de vos téléchargements. L'option supplémentaire « Réduire automatiquement en cas de lancement à partir du navigateur » est installée par défaut si vous activez « Réduire la zone de notification ».
