Amazon MP3ダウンローダー

Amazon MP3ダウンローダーは、アルバムや複数の曲をまとめてダウンロードする際に必要となるツールです。初めてMP3をご購入いただく場合には、Amazon MP3ダウンローダーをインストールするようにメッセージが表示されます。Amazon MP3ダウンローダーを使ってダウンロードした曲は、iTunesまたはWindows Media Player（WMP）のライブラリに自動的にインポートされます。

システム要件

Amazon MP3ダウンローダーは以下のOS（オペレーティングシステム）でご利用いただくことができます。

  • Windows： Windows XP 、 VistaまたはWindows 7
  • Mac：Mac OS X 10.4以上
  • Linux： Debian 5、Fedora 12、Open SUSE 11.2 または Unbuntu 9.10

ブラウザ

Amazon MP3ダウンローダーはお使いのPCでデフォルトに設定されているブラウザでご利用ください。他のブラウザで使用すると、問題が発生することがあります。

例えば、デフォルトのブラウザがInternet Explorerの場合、それ以外のブラウザ（ Firefoxなど ）でAmazon MP3ダウンローダーをインストールしても、インストール後にはInternet Explorerが起動するため、正常に商品をダウンロードすることができなくなります。

ブラウザの設定

  • 任意のブラウザをデフォルトブラウザとして設定する：お使いのブラウザのヘルプをご確認のうえ、任意のブラウザをデフォルトブラウザとして設定してください。設定が完了したらデフォルトのブラウザで、Amazon MP3ダウンローダーをインストールしてください。
  • Amazon MP3ダウンローダーを任意のブラウザで利用可能にする：Amazon MP3ダウンローダーがすでにインストールされている場合は、インストールに使用したブラウザでこちらのページをご確認のうえ「トラブルシューティング」の項目をご覧ください。

インストール

インストール方法

  1. こちらをクリックしてAmazon MP3ダウンローダーのインストール画面にアクセスします
  2. お使いのOSを選択のうえ「今すぐインストール」ボタンをクリックします（OSは自動的に検知されます）
  3. 画面の指示に従ってインストールを完了します

無料の曲でAmazon MP3ダウンローダーをテスト

Amazon MP3ダウンローダーのインストール後、まず、テスト用の曲（無料）で、正常にダウンロードができるかどうか確認します。初回のご購入時には、テスト用の曲をダウンロードした後に、実際に購入した曲をダウンロードします。

AMZファイルはAmazon MP3ダウンローダーを自動的に起動し、MP3のダウンロードを開始するためのファイルです。AMZファイルをダウンロードした際は、必ず「開く」を選択してください。

テスト用の曲のダウンロードが完了すると、iTunesまたはWindows Media Playerが自動的に起動します。その後、ダウンロードした曲を自動的にインポートしますので、正しく再生できるかどうかご確認ください。メディアプレーヤーへのインポートについて詳しくはこちらをご覧ください。

利用方法

MP3は、ご注文確定後にダウンロードしていただく必要があります。

Amazon MP3ダウンローダーの起動

Amazon MP3ダウンローダーはMP3のご注文を確定した後に自動的に起動します。もしAMZファイルのダウンロードを中断したりキャンセルした場合、または AMZファイルについて「保存する」を選択した場合には、手動でAmazon MP3ダウンローダーを起動してください。

注意：MP3のダウンロードはご注文確定後の限られた時間内でのみ可能です。AMZファイルをPCに保存した後は、すぐにAmazon MP3ダウンローダーをご利用のうえ、ファイルを開いてください。ご注文確定後、一定時間（約3時間）が経過するとファイルが無効になりますのでご注意ください。

Amazon MP3ダウンローダーを手動で起動する場合は、以下の情報を参照してください。

  • Windows：すべてのプログラム＞Amazon＞Amazon MP3ダウンローダー
  • Mac：「アプリケーション」フォルダから「Amazon MP3ダウンローダー」を選択します。
  • Linux：Amazon MP3ダウンローダーは「/usr/bin/amazonmp3」にインストールされています。「アプリケーション」メニュー内の「インターネット」サブメニューにアイコンが表示されます。

操作ボタン

ダウンロード中は、ダウンロードのステータスによって以下の操作ボタンが画面上に表示されます。操作できないボタンは、グレーで表示されます。

  • ダウンロードを一時停止：曲やアルバムのダウンロードを一時停止します。一時停止した場合は、3時間以内にダウンロードを再開してください。ダウンロードの再開時に問題が発生した場合は、カスタマーサービスにご連絡ください。
  • ダウンロードを再開：ダウンロードを一時停止している場合に、ダウンロードを再開します。
  • ダウンロード済みファイルをクリア：すでにダウンロードが完了した曲やアルバムをダウンロードウインドウの表示から消去することができます。ダウンロードを一時停止した場合やキャンセルした場合にこのボタンをクリックしても、ダウンロードリストから商品は削除されません。なお、Amazon MP3ダウンローダーの終了時に、ダウンロード済みの曲やアルバムは自動的に一覧から消去されます。
  • 選択した曲を再生：デフォルトに設定されているメディアプレーヤーを起動し、選択した曲を再生します。アルバムを選択した場合は、Amazon MP3フォルダが開き、再生したい曲をその中から選ぶことができます。
  • ダウンロードしたアイテムを確認：ご利用可能なオプションはお使いのOSによって異なります。このボタンをクリックすると、MP3ダウンロードストアで購入した曲を保存するデフォルトのフォルダを開きます。デフォルトのフォルダは「Amazon MP3」というフォルダ名で保存されています。このフォルダは通常、「マイミュージック」フォルダ（Windows）や「ミュージック」フォルダ（Mac）の中にあります。

iTunesやWindows Media Playerへのインポート

お使いのPCにiTunesまたは Windows Media Player（WMP）がインストールされている場合、Amazon MP3ダウンローダーは、デフォルトに設定されているメディアプレーヤーを自動的に起動し、そのライブラリに曲をインポートします。どちらのメディアプレーヤーもインストールされていない場合は、デフォルトに設定されているダウンロード先フォルダにファイルを保存します。

なお、メディアプレーヤーのライブラリへの自動インポート機能は、Linux版のAmazon MP3ダウンローダーではご利用いただけませんのでご了承ください。

インポート機能設定の変更方法

  1. Amazon MP3ダウンローダーを起動します
  2. メニューの「ファイル」から「詳細設定」を選択します
  3. 「メディアライブラリ」の項目からご希望のメディアプレーヤーを選択します。インポート機能を使用しない場合は、「選択しない」を選択します

iTunesをご利用いただく際の注意

インストールされたiTunesのバージョンによっては、iTunesへ曲をインポートする際に、まれに問題が発生することがあります。その場合、Amazon MP3ダウンローダーの画面上に、曲をインポートできなかった旨のメッセージが表示されます。この問題を解決するには、www.apple.com/jp/itunes/　にアクセスのうえ、最新バージョンのiTunesをインストールしてください。インストール後は、PCを再起動し、iTunesを起動した後、Amazon MP3ダウンローダーを起動し、iTunesに曲をインポートしてください。

ファイルの保存先

Amazon MP3ダウンローダーを使って曲をダウンロードした場合、デフォルト設定では、曲は「Amazon MP3」フォルダに保存されます。このフォルダは通常、「マイミュージック」フォルダ（Windows）や「ミュージック」フォルダ（Mac）の中にあります。ファイルの保存先は以下の手順に従って変更することができます。

ファイルの保存先の変更方法：

  1. Amazon MP3ダウンローダーを起動します
  2. メニューの「ファイル」から「詳細設定」を選択します
  3. 「ダウンロード先のフォルダ」の項目で「変更」ボタンをクリックします
  4. 一覧からフォルダを選択します
  5. 選択したフォルダ内に新しいフォルダを作成したい場合は、「新しいフォルダを作成する」ボタンをクリックします
  6. 「OK」ボタンをクリックします

Amazon MP3ダウンローダーを使わずに曲をダウンロードした場合、「.mp3」という拡張子からファイルを探すか、ブラウザでデフォルトとして指定されているファイル保存先からファイルを探してください。ファイルの検索方法について詳しくは、お使いのOSのヘルプをご確認ください。

メニュー

Amazon MP3ダウンローダーのメニューでは以下の項目をご利用いただくことができます。

  • ファイル/詳細設定：詳細設定画面を開き、 Amazon MP3ダウンローダーの各種設定を変更することができます。
  • ファイル/終了：Amazon MP3ダウンローダーのアプリケーションを終了します。
  • ヘルプ/ヘルプ：ブラウザのウインドウでAmazon MP3ダウンローダーのヘルプを開きます。
  • ヘルプ/概要：Amazon MP3ダウンローダーのバージョン情報を表示します。

その他の設定

Amazon MP3ダウンローダーの設定を変更するには、メニューから「ファイル」を選択し、「詳細設定」をクリックしてください。詳細設定では以下の設定を変更することができます。

  • メディアプレーヤー：ダウンロードした曲をお使いのメディアプレーヤーのライブラリに自動的にインポートしたい場合は、このオプションのプルダウンメニューからご希望のメディアプレーヤーを選択してください。現在サポートしているメディアプレーヤーはiTunesとWindows Media Playerの2種類です。「選択しない」を選択すると、ダウンロードした曲は「Amazon MP3」フォルダに保存されます。この項目はLinux版ではご利用いただけませんのでご了承ください。
  • プログラムのアップデートを自動的にチェックする：このオプションを有効にすると、Amazon MP3ダウンローダーの起動時に、プログラムの重要なアップデートがないかを自動的にチェックすることができます。
  • システムトレイに最小化する（Windows版のみ）：このオプションを有効にすると、Amazon MP3ダウンローダーをWindowsのシステムトレイのタスクバーに最小化することができます。「バルーンヒントを表示する」をあわせて有効にすると、 Amazon MP3ダウンローダーの状況についてバルーンヒントを表示することができます。また、このオプションを有効にした場合、「ブラウザから起動した際に自動的に最小化する」の追加オプションもデフォルトで有効になります。
