Systemvoraussetzungen
  • PC: Windows XP, Vista oder 7
  • Mac: Mac OS X 10.4 oder höher
  • Linux: Debian 5, Fedora 11, Open SUSE 11.1 und Ubuntu 9.04

Browservoraussetzungen

Amazon MP3-Downloads sind mit allen gängigen Webbrowsern kompatibel. Darüber hinaus ist es jedoch notwendig, Ihr System und Ihren Browser so einzustellen, dass das Herunterladen von MP3-Dateien problemlos möglich ist.

Einkaufen mit Ihrem Standardbrowser
Der Amazon MP3-Downloader greift auf den von Ihrem System voreingestellten Browser zu. Am Anfang oder Ende eines Downloads kann es zu Problemen kommen, wenn Sie beim Einkauf einen anderen Browser verwenden.

Wenn Sie den Amazon MP3-Downloader mit einem anderen Browser (zum Beispiel Firefox) installieren, Ihr voreingestellter Browser aber beispielsweise der Internet Explorer ist, startet Ihr voreingestellter Browser nach Installation des Amazon MP3-Downloaders. Dies bedeutet, dass die Einkäufe über den anderen Browser nicht abgeschlossen werden können.

Wie Sie den Browser auswählen, den Sie für Ihre Einkäufe nutzen möchten:

  • Richten Sie den von Ihnen gewünschten Browser als Standardbrowser ein: Informationen dazu finden Sie in der Hilfefunktion Ihres Browsers. Anschließend bitte den Amazon MP3-Downloader installieren.

  • Den Amazon MP3-Downloader für den aktuellen Browser aktivieren: Wenn der Amazon MP3-Downloader bereits auf Ihrem System installiert ist, gehen Sie bitte in Ihrem gewünschten Browser auf die Installierungsseite des Downloaders unter http://www.amazon.de/gp/dmusic/help/amd.html/ref=sv_dmusic_3. Folgen Sie dort bitte den Hinweisen unter "Hilfestellung".

Die Einstellungen Ihres Browsers anpassen
Manchmal kann es an den Einstellungen Ihres Browsers liegen, dass Ihre MP3-Dateien nicht auf Ihren Computer heruntergeladen werden können. Genauere Hinweise zum Anpassen Ihrer Browsereinstellungen finden Sie in Ihrem Browsermenü unter "Hilfe".

  • Cookies zulassen: Amazon MP3-Downloads und der Amazon MP3-Downloader nutzen Cookies, um Ihre Einkäufe und das Herunterladen Ihrer MP3-Dateien zu steuern. Bitte stellen Sie Ihren Browser so ein, dass er Cookies von Amazon.de akzeptiert.

    Wenn Sie Cookies nicht oder nur temporär für jede Sitzung zulassen, werden Sie in der Regel nach jedem Einkauf aufgefordert, den Amazon MP3-Downloader zu installieren.

  • Pop-Up-Blocker deaktivieren: Bitte stellen Sie Ihren Browser so ein, dass er Pop-Ups von Amazon.de akzeptiert. Amazon MP3-Downloads und der Amazon MP3-Downloader greifen auf Ihren voreingestellten Browser zu und öffnen neue Fenster für den Einkauf und das Herunterladen der MP3-Dateien.

  • Downloads von Amazon.de zulassen: Bitte setzen Sie Amazon.de auf die Liste vertrauenswürdiger Websites und lassen Sie Downloads von Amazon.de zu.

  • Download-Mitteilungen aktivieren: Die Sicherheitseinstellungen Ihres Browsers können Downloads verhindern, ohne dass Sie einen entsprechenden Hinweis erhalten. Bitte überprüfen Sie die Sicherungseinstellungen Ihres Browsers, ob die Download-Mitteilungen aktiviert sind. So wissen Sie, ob Ihr Browser Downloads verhindert.

Amazon MP3-Downloader installieren

Wenn Sie Ihre erste MP3-Datei kaufen, werden Sie automatisch aufgefordert, den Amazon MP3-Downloader zu installieren.

So installieren Sie den Amazon MP3-Downloader:

  1. Gehen Sie auf die Installationsseite des Amazon MP3-Downloaders.
  2. Klicken Sie auf den Schaltknopf Jetzt herunterladen für Ihr Betriebssystem. (Ihr Betriebssystem wird normalerweise automatisch identifiziert.)
  3. Folgen Sie den Anweisungen zur Installation auf der Installationsseite des Amazon MP3-Downloaders.

Amazon MP3-Downloader nutzen

Alle MP3-Musikdateien müssen zum Zeitpunkt des Einkaufs heruntergeladen werden. Der Amazon MP3-Downloader speichert sie in einem dafür vorgesehenen Verzeichnis auf Ihrem Computer und kann sie automatisch in Ihre iTunes-Bibliothek oder in Ihren Windows Media Player importieren.

Den Amazon MP3- Downloader öffnen
Der Amazon MP3-Downloader öffnet sich automatisch, sobald Sie bei Amazon MP3-Downloads einkaufen. Er muss nur dann manuell von Ihrem Computer aus gestartet werden, wenn Sie das Herunterladen einer MP3-Datei unterbrechen, abbrechen oder die Datei auf Ihren Computer speichern statt sie mit Hilfe des MP3-Downloaders zu öffnen.

Hinweis: Das Herunterladen ist nur für kurze Zeit nach dem Einkauf möglich. Wenn Sie die .amz-Datei auf Ihrem Computer gespeichert haben, sollten Sie gleich nach dem Speichern den Amazon MP3-Downloader öffnen. Wenn Sie mit dem Herunterladen warten, ist es möglich, dass die Datei nicht mehr verfügbar ist.

So öffnen Sie den Amazon MP3-Downloader, wenn Sie einen unvollständigen Download abschließen möchten:

  • Windows: Alle Programme > Amazon > Amazon MP3-Downloader
  • Mac: Wählen Sie "Amazon MP3-Downloader" aus Ihrer Programmliste.
  • Linux: Die Anwendung ist installiert als /usr/bin/amazonmp3 und ein Symbol erscheint unter "Anwendung" im Unterordner "Internet".

Verfügbare Befehle des Amazon MP3-Downloaders
Wenn Musik-Dateien heruntergeladen werden, erscheinen die unten stehenden Befehle -- abhängig davon, welche Dateien heruntergeladen werden und je nach aktuellen Status des Downloads. Nicht verfügbare Befehle sind grau hinterlegt.

  • Downloads anhalten: Hält das Herunterladen von Alben oder Songs an. Diese Artikel bleiben im Amazon MP3-Downloader, auch wenn Sie die Anwendung schließen und neu starten. Jedoch raten wir davon ab, das Herunterladen für mehr als 3 Stunden zu unterbrechen. Wenn Sie Schwierigkeiten haben, die MP3-Dateien nach Unterbrechen des Downloads herunterzuladen, kontaktieren Sie bitte unseren Kundenservice.

  • Downloads fortsetzen: Das Herunterladen Ihrer Dateien wird fortgesetzt.

  • Abgeschlossene Downloads entfernen: Heruntergeladene Alben und Songs werden aus dem Download-Fenster entfernt. Wenn das Herunterladen nicht zu Ende geführt wurde, weil Sie den Download gestoppt oder unterbrochen haben, werden diese Songs durch Auswählen dieser Option nicht von der Download-Liste entfernt. Alle Downloads, die vollständig heruntergeladen wurden, werden entfernt, wenn Sie die Anwendung schließen.

  • Download-Ordner ansehen: Ihr voreingestellter Media-Player öffnet sich und spielt den von Ihnen ausgewählten Song. Wenn ein Album ausgewählt wurde, öffnet sich bei dieser Option das Verzeichnis "Amazon MP3", sodass Sie die verfügbaren Songs auswählen können.

  • Downloads-Verzeichnis öffnen: Welche Option verfügbar ist, hängt von Ihrem Betriebssystem ab.
    • Downloads-Ordner ansehen (Windows)
    • Im Finder anzeigen (Mac)
    • Download-Verzeichnis ansehen (Linux)

    Öffnet das voreingestellte Downloadverzeichnis, in dem Ihre MP3-Käufe auf Ihrem Computer gespeichert werden. Das Verzeichnis auf Ihrem PC lautet "Amazon MP3". Sie finden es unter "Meine Musik" (Windows) beziehungsweise "Musik" (Mac). Heruntergeladene Songs finden.

Auf iTunes oder Windows Media Player importieren

Wenn Sie iTunes oder Windows Media Player (ab Version 10) auf Ihrem PC installiert haben, startet der Amazon MP3-Downloader automatisch die jeweilige als Standardplayer ausgewählte Anwendung und importiert Ihre Songs in Ihre Medien-Bibliothek. Anderenfalls lädt der Amazon MP3-Downloader Ihre MP3-Dateien in Ihr Standardverzeichnis für Downloads.

Den Windows Media Player (für Mac and PC) können Sie auf der Seite von Microsoft herunterladen, iTunes können Sie auf der Website von Apple herunterladen (für Mac und PC).

Hinweis: Die Funktion, Dateien in eine Medien-Bibliothek zu importieren, steht für die Linux-Version des Amazon MP3-Downloaders nicht zur Verfügung.

So bestimmen Sie die gewünschte Importfunktion:

  1. Öffnen Sie den Amazon MP3-Downloader.
  2. Wählen Sie "Datei > "Einstellungen" aus dem Menü des Amazon MP3-Downloaders.
  3. Wählen Sie den gewünschten Standardplayer unter "Medien-Bibliothek". Wählen Sie "Nichts tun", wenn Sie die Importfunktion nicht nutzen möchten.

Die Optionen für "Zu iTunes hinzufügen" oder "Zu Windows Media Player hinzufügen" erscheinen nur, wenn die jeweilige Anwendung auf Ihrem Computer installiert ist.

Ein Hinweis für iTunes-Nutzer
In seltenen Fällen kann der Amazon MP3-Downloader aufgrund der vorliegenden Version keine Songs auf iTunes importieren. Der Amazon MP3-Downloader gibt einen entsprechenden Hinweis rechts oben auf Ihrem Bildschirm. In diesem Fall empfehlen wir, auf die neueste Version von iTunes über die Website von Apple zu installieren.

Danach starten Sie Ihren Computer bitte neu. Starten Sie iTunes und akzeptieren Sie die Nutzungsbedingungen von Apple, bevor Sie über den Amazon MP3-Downloader Songs auf iTunes importieren.

Wenn Sie das Download-Verzeichnis Ihres Amazon MP3-Downloaders zu Ihren iTunes-Einstellungen hinzufügen möchten:

  • Die Anwendung iTunes starten und "Ordner zu Mediathek hinzufügen" aus dem Menü auswählen.
  • Wählen Sie dann das Verzeichnis, in das Ihre Musik geladen werden soll. Das Standardverzeichnis lautet "Amazon MP3" und befindet sich bei Windows im Ordner "Meine Musik" und bei einem Mac im Ordner "Musik".

Heruntergeladene Songs finden

Der Amazon MP3-Downloader speichert die heruntergeladenen Songs standardmäßig auf Ihrem Computer im Verzeichnis "Amazon MP3". Dieses finden Sie als Windows-Nutzer im Ordner "Meine Musik" und als Mac-Nutzer im Verzeichnis "Musik". Wir empfehlen, dies beizubehalten, Sie können diese Einstellung jedoch auch ändern.

Wenn Sie den Amazon MP3-Downloader nicht zum Speichern Ihrer Musik verwendet haben, finden Sie die Dateien, indem Sie in Ihrem Computer nach ".mp3" suchen oder indem Sie in Ihrem Browser den üblichen Speicherort aufrufen.

Um Ihre Videos im Windows Media Player zu finden, klicken Sie bitte auf den nach unten gerichteten Pfeil unterhalb "Library" in der Mitte der Applikation und wählen Sie im Drop-Down-Menü "Video". Sie sollten nun die mit dem Amazon MP3-Downloader heruntergeladenen Videos angezeigt bekommen. Digitale Booklets werden nicht in den Windows Media Player heruntergeladen, sondern in Ihrem voreingestellten Ordner für Downloads gespeichert.

So ändern Sie den voreingestellten Speicherort:

  1. Starten Sie den Amazon MP3-Downloader.
  2. Wählen Sie aus dem Menü des Amazon MP3-Downloaders "Datei > Einstellungen".
  3. Klicken Sie den Schaltknopf Ändern im Bereich "Verzeichnis auswählen".
  4. Wählen Sie ein Verzeichnis aus der Übersicht.
  5. Wenn Sie in dem gewählten Verzeichnis ein Unterverzeichnis erstellen möchten, klicken Sie auf den Schaltknopf Neues Verzeichnis erstellen.
  6. Am Schluss klicken Sie bitte auf OK.

Optionen beim Amazon MP3-Downloader

Die folgenden Optionen finden Sie im Menü Ihres MP3-Downloaders:

  • Datei > Einstellungen: Öffnet das Fenster mit den Einstellungen für den Amazon MP3-Downloader. Einstellungen beim Amazon MP3-Downloader ändern."

  • Datei > Beenden: Schließt den Amazon MP3-Downloader.

  • Hilfe > Hilfe anzeigen: Öffnet die Hilfe für den Amazon MP3-Downloader in einem Browserfenster.

  • Hilfe > Nach neuester Version suchen: Überprüft, ob eine neuere Version des Amazon MP3-Downloaders verfügbar ist.

  • Hilfe > Kundenservice kontaktieren: Kundenservice: Öffnet das Kontaktformular.

  • Hilfe > Über: Zeigt Informationen zur Version des Amazon MP3-Downloaders an.

Einstellungen beim Amazon MP3-Downloader ändern

Um auf die Einstellungen des Amazon MP3-Downloaders zuzugreifen, wählen Sie bitte "Datei" aus dem Menü und klicken Sie dann auf die Option "Einstellungen". Folgende Einstellungen können Sie verändern:

  • Download-Verzeichnis: Hiermit können Sie neu festlegen, wo Ihre gespeicherten Downloads aus dem Standardverzeichnis gespeichert werden sollen. Klicken Sie auf den Schaltknopf Ändern, um ein Verzeichnis auszuwählen oder ein neues zu erstellen und bestätigen Sie mit OK.

  • Medien-Bibliothek: Wählen Sie einen Media-Player aus dem Menü, wenn Ihre Einkäufe automatisch zu Ihrer Medien-Bibliothek hinzugefügt werden soll. Die Media-Player, die Sie derzeit für den Amazon MP3-Downloaders nutzen können, sind iTunes und derWindows Media Player. Beim Auswählen von "Nichts tun" werden Ihre Musikdateien im "Amazon MP3"-Verzeichnis abgelegt. Sie können dann manuell zu Ihrer ausgewählten Medien-Bibliothek hinzugefügt werden. Für Linux-Betriebssysteme ist diese Option nicht verfügbar.

  • Automatisch nach Updates suchen: Beim Anklicken dieses Kästchens sucht der Amazon MP3-Downloader nach wichtigen Aktualisierungen, sobald Sie die Anwendung öffnen.

  • Minimieren und im Info-Bereich anzeigen (Nur Windows): Der MP3-Downloader wird minimiert und im Info-Bereich der Windows-Taskleiste angezeigt. Bei dieser Option können Sie auch "Tipps anzeigen" auswählen. So werden Ihnen Hinweise zur Aktivität des Amazon MP3-Downloaders angezeigt. Die zusätzliche Option "Automatisch verkleinern, wenn durch Browser gestartet" wird automatisch ausgewählt, wenn Sie die Option "Minimieren und im Info-Bereich anzeigen" auswählen.
