System Requirements
  • PC: Windows XP, Vista or 7
  • Mac: Mac OS X 10.4 or higher
  • Linux: Debian 5, Fedora 11, Open SUSE 11.1, and Ubuntu 9.04

Browser Requirements

While the Amazon MP3 is compatible with the most commonly used Web browsers, you will need to configure your system and browser settings to allow downloads from Amazon.com to complete successfully.

Shopping With Your Default Browser
The Amazon MP3 Downloader works with your system's default Web browser. You may encounter issues initiating or completing your download if you use an alternate browser to shop Amazon MP3.

For example, if you install the Amazon MP3 Downloader using an alternate browser (e.g. Firefox) that is not your system's default browser (e.g. Internet Explorer), your default browser will launch after the Amazon MP3 Downloader installs and you will not be able to complete purchases from your alternate browser.

To enable your preferred browser:

  • Set preferred Web browser as system default: Please refer to your browser's Help section for information on designating it as your system's default browser. After your system is set to the preferred default browser, install the Downloader.

  • Enable Amazon MP3 Downloader for current browser: If the Amazon MP3 Downloader is already installed on your system, visit the Amazon MP3 Downloader install page at http://www.amazon.com/gp/dmusic/help/amd.html/ using your preferred Web browser and follow the instructions in the "Troubleshooting" section.

Adjusting Your Browser Settings
In certain cases, settings on your browser may be preventing your files from being downloaded to your computer. Please refer to your browser's Help section for specific instructions on adjusting your browser settings.

  • Enable Cookies: The Amazon MP3 Downloader uses cookies to work with Amazon.com to confirm your purchases and manage your Amazon MP3 file downloads. You will need to enable your browser to accept cookies from Amazon.com to successfully download files.

    If you do not enable cookies or only allow cookies temporarily by session, you may be asked to install the Amazon MP3 Downloader after each purchase. Always accepting cookies from Amazon.com will eliminate this type of issue.

  • Disable Pop-Up Blockers: You should adjust your browser settings to allow pop-up windows from Amazon.com. The Amazon MP3 Downloader works with your default browser and will launch new windows to help you complete purchases and downloads.

  • Enable Downloads from Amazon.com: You may need to add Amazon.com to your list of trusted sites or allow downloads from Amazon.com if your browser displays a security warning.

  • Enable Download Notifications: In certain cases, your browser's security settings may be preventing file downloads without notifying you that a download is being attempted, particularly if you use Internet Explorer. Check your browser's security settings to enable automatic prompting for file downloads so you know if your browser is blocking downloads.

Installing the Amazon MP3 Downloader

We'll automatically prompt you to install the Amazon MP3 Downloader when you make your first MP3 purchase.

To install the Amazon MP3 Downloader:

  1. Click here to visit the Amazon MP3 Downloader install page.
  2. Click the Download Now button for your operating system. (Your OS should be detected automatically.)
  3. Follow the "Detailed Installation Instructions" on the Amazon MP3 Downloader install page.

Testing The Downloader With Your Free Song
We offer you a Free song after the Amazon MP3 Downloader installation completes so you can test the Amazon MP3 Downloader and make sure it is working. If you are installing the Downloader to complete your first MP3 purchase, the music you bought will download after the Free song.

Be sure to "Open" the .amz file if prompted and configure your settings to always use the Amazon MP3 Downloader to "Open" .amz files. The .amz file type activates the Downloader and initiates your MP3 file downloads.

After your free song downloads, the Amazon MP3 Downloader will open your iTunes or Windows Media Player (if you have either installed) and will automatically import the downloaded song so you can start listening to it. See Importing into iTunes or Windows Media Player for more information.

Using the Amazon MP3 Downloader

All content you buy from Amazon MP3 must be downloaded at the time of purchase. The Amazon MP3 Downloader streamlines the process of downloading and organizing the content you purchase by enabling you to download entire albums, queue up multiple song purchases, and automatically import content into iTunes or Windows Media Player.

Launching the Amazon MP3 Downloader
The Downloader launches automatically each time you make a purchase. You will only need to launch the Amazon MP3 Downloader manually from your computer if you pause or cancel a download, or "Save" the .amz file to your computer instead of opening it with the Amazon MP3 Downloader.

Note: Downloads are only available for a short time after purchase. If you saved the .amz file to your computer, you should "Open" it using the Downloader application immediately after saving it. If you wait to complete your download, it may expire and no longer be available.

Here's where to find and launch the Amazon MP3 Downloader application if you need to resume an incomplete download:

  • Windows: All Programs > Amazon > Amazon MP3 Downloader
  • Mac: Select "Amazon MP3 Downloader" from your Applications folder.
  • Linux: The application is installed as /usr/bin/amazonmp3 and an icon appears in the "Applications" menu in the "Internet" submenu.

Amazon MP3 Downloader Command Buttons
When a download is in progress, the commands listed below will be available depending on the items downloading and their status. If a command is not available for the selected item, the button will be grayed out.

  • Pause Downloads: Pauses the download of all albums or songs if any downloads are in progress. Paused items will be available in the Amazon MP3 Downloader even if you close and restart the application, but we do not recommend pausing downloads unless you plan to resume the downloads within 3 hours. If you have difficulty resuming paused downloads, please contact Customer Support.

  • Resume Downloads: Resumes downloading any previously paused songs.

  • Clear Completed: Removes downloaded albums and songs from the download window. If a download was incomplete because you chose to cancel or pause it, selecting this option will not remove those items from your download list. All completed downloads will clear when you close the application.

  • Open Selected: Launches your default media player and plays the selected song. If an album is selected, choosing this option will open the Amazon MP3 folder so you can choose from available content.

  • "View Downloads": The available option varies depending on your operating system.
    • View Download Folder (Windows)
    • Reveal in Finder (Mac)
    • View Download Folder (Linux)

    Opens the default download folder where your Amazon MP3 purchases are stored on your computer. The default folder is "Amazon MP3" on your computer. This will typically appear in your "My Music" folder (Windows) or your "Music" folder (Mac). See Locating Downloaded Songs for more information.

  • Downloading MP3 files from Your Media Library

    Downloading MP3 files from Your Media Library

    1. Go to Your Media Library
    2. Log in, if prompted.
    3. Click on the "Downloads" tab.
    4. Hover over the arrow on the "Downloads" tab and choose "Amazon MP3" from the pop-up menu. (This will filter the results so only your music downloads are displayed).
    5. Find the album in the list and click on the "Download" button to display songs available for download. If you purchased only one song, Your Media Library will still display your purchases by the album the song is from.
    6. On the album detail page, click the "Download" link next to each song available to start the download.
    7. If you have the Amazon MP3 Downloader installed, choose the option to open the .amz file when prompted.
    8. If you do not have the Amazon MP3 Downloader installed, choose "Save" when prompted to save the file.

    Importing into iTunes or Windows Media Player

    If you have iTunes or Windows Media Player installed on your computer, the Amazon MP3 Downloader will automatically launch the application designated as the default media player and import your content into your library. If neither application is installed, the Amazon MP3 Downloader will simply save the downloads to your default Download Folder.

    Note: The media library import function is not available for the Linux version of the Amazon MP3 Downloader.

    To set the default importing behavior:

    1. Launch the Amazon MP3 Downloader.
    2. Select "File/Preferences . . ." from the Amazon MP3 Downloader menu.
    3. Choose the desired default player under the "Media Library" section. Choose "Do nothing" if you do not wish to use the import function.

    The options for "Add it to iTunes" or "Add it to Windows Media Player" will only appear if those applications are installed on your computer system.

    A Note about Using iTunes
    In certain rare cases, the Downloader may fail to import songs into iTunes due to potential issues with the installed version of iTunes. The Downloader should provide a message in the right-hand corner of your screen that it failed to add songs to iTunes. If this occurs, we recommended you visit www.iTunes.com and install the most recent version of the application. After reinstalling iTunes, please restart your computer, launch iTunes, and accept Apple's terms of use before attempting to use the Amazon MP3 Downloader to import songs to iTunes.

    You may also wish to add your Amazon MP3 Downloader's Download Folder to your iTunes settings. To do this, launch iTunes and select the "Add Folder to Library" option from the menu and select the folder to which your songs download. The default folder is called "Amazon MP3" and resides in the "My Music" folder in Windows or the "Music" folder on a Mac.

    Locating Downloadeds

    When the Amazon MP3 Downloader saves content to your computer, it saves them to a folder called "Amazon MP3" by default. This folder is located within the "My Music" folder in Windows or the "Music" folder on a Mac. We recommend using the default folder, but you can change this setting if you wish.

    If you did not use the Amazon MP3 Downloader to save your content, you can find files by searching for ".mp3" on your computer or by checking the default save location for your Web browser.

    To find your videos in Windows Media Player, click on the downward facing arrow below “Library” in the top center of the application and select “”Video” form the drop down. You should now see the videos downloaded by the Amazon MP3 Downloader. Digital booklets are not downloaded into Windows Media Player and are instead saved to your default download folder.

    To change the default Amazon MP3 Downloader save location:

    1. Launch the Amazon MP3 Downloader.
    2. Select "File/Preferences . . ." from the Amazon MP3 Downloader menu.
    3. Click the Change button in the Download Folder section.
    4. Select a folder from the list.
    5. If you wish to create a subfolder inside the selected folder, click the Make New Folder button.
    6. Click the OK button when finished.

    Amazon MP3 Downloader Menu Options

    The following options are available in the Amazon MP3 Downloader menus:

    • File/Preferences: Opens the Preferences window where you can adjust the Amazon MP3 Downloader settings. More information is available below under "Changing Downloader Preferences."

    • File/Exit: Closes the Amazon MP3 Downloader application.

    • Help/Show Help: Opens the Amazon MP3 Downloader application Help page in a Web browser window.

    • Help/Check for Updates: Checks Amazon.com for a newer version of the Amazon MP3 Downloader.

    • Help/Contact Customer Service: Customer Service: Opens the Customer Service contact us page.

    • Help/About: Displays version information for the Amazon MP3 Downloader.

    Changing Amazon MP3 Downloader Preferences

    To access the Amazon MP3 Downloader Preferences, choose "File" from the menu and then click on the "Preferences" option. Here's a list of the settings you can customize:

    • Download Folder: This option allows you to change the location of your stored downloads from the default directory. Click the Change button to choose or create a new folder. Click the OK button when you are finished.

    • When a track finishes downloading: Select a supported media player from the pull-down menu if you want your purchases added to your media library automatically. The media players currently supported are iTunes and Windows Media Player. If you choose to "Do Nothing," your music files will be stored in the "Amazon MP3" folder and can be added to your desired media library manually. This option is not available for Linux operating systems.

    • Automatically check for program updates: Clicking the box next to this option will automatically enable the Amazon MP3 Downloader to check for critical updates when you open the application.

    • Minimize to notification tray (Windows Only): Selecting this option minimizes the Amazon MP3 Downloader into your Windows system tray to on your taskbar. If you choose this option you can also enable "Show Balloon Tips" so that you will see notifications about Amazon MP3 Downloader activity. The additional option "Automatically minimize when started from browser" is selected by default when you enable the "Minimize to notification tray" option.
